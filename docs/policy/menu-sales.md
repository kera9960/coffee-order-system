# 메뉴 판매 정책

## 규칙

### 메뉴 상태

- 판매 중인 메뉴의 상태는 `ON_SALE`이다.
- 판매가 중지된 메뉴의 상태는 `STOPPED`이다.

### 판매 재개

- `STOPPED` 상태인 메뉴는 다시 판매할 수 있으며, 판매를 재개하면 상태를 `ON_SALE`로 변경한다.
- 판매가 재개된 메뉴는 일반 메뉴 목록과 인기 메뉴 조회 결과에 다시 포함될 수 있으며 신규 주문에도 포함할 수 있다.


### 고객 노출

- 일반 메뉴 목록에는 `ON_SALE` 메뉴만 노출한다.
- `STOPPED` 메뉴는 일반 메뉴 목록에 노출하지 않는다.
- `STOPPED` 메뉴는 인기 메뉴 조회 결과에도 노출하지 않는다.

### 신규 주문

- 신규 주문에는 `ON_SALE` 메뉴만 포함할 수 있다.
- 주문 처리 시점에 `STOPPED` 상태인 메뉴는 주문할 수 없다.
- 한 주문에 판매 중지 메뉴가 하나라도 포함되어 있으면 해당 주문 전체를 완료할 수 없다.

### 기존 주문 보존

- 판매 중지된 메뉴는 기존 주문 내역 보존을 위해 즉시 삭제하지 않는다.
- 메뉴가 판매 중지되어도 이미 완료된 주문 내역은 유지한다.
- 메뉴 상태, 이름 또는 가격이 변경되어도 과거 주문 정보는 변경하지 않는다.

## 근거 / 배경

판매할 수 없는 메뉴가 고객에게 노출되거나
신규 주문에 포함되는 것을 방지해야 한다.

동시에 판매가 중지된 이후에도 과거 주문 내역을 정상적으로 조회할 수 있어야 하므로
메뉴 데이터를 즉시 삭제하지 않는다.

## 적용 대상

- 판매 중인 메뉴 목록 조회
- 주문 생성
- 주문 목록 조회
- 주문 상세 조회
- 인기 메뉴 집계
- 인기 메뉴 조회
